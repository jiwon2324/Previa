# Previa

대학생의 학습 일정, 강의 자료, 퀴즈, 플래시카드와 AI 학습 도우미 경험을 하나의 모바일 화면에 구성한 프론트엔드 프로토타입입니다.

## 주요 기능

- 학습 일정과 주차별 진도 관리
- 강의 자료 검색 및 정리
- 퀴즈, 오답 노트, 플래시카드
- AI 학습 도우미 UI
- 모바일 앱 형태의 반응형 인터페이스

## 파일 구성

- `index.html`: 화면 구조
- `style.css`: 디자인과 레이아웃
- `app.js`: 화면 전환 및 상호작용

## 실행 방법

저장소를 내려받은 뒤 `index.html`을 브라우저에서 열면 됩니다.

> 공개 저장소에는 API 키가 포함되어 있지 않습니다. 실제 API 연동 시 키를 프론트엔드 코드에 직접 넣지 말고 서버 환경 변수로 관리하세요.
